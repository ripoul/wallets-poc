# Google & Apple Wallets Integration POC

This repository is a Proof of Concept (POC) for integrating both Google Wallet and Apple Wallet into your applications. It demonstrates how to create, manage, and distribute digital passes such as loyalty cards, tickets, or coupons using the respective APIs from Google and Apple.

## Features

- Generate and manage Google Wallet passes
- Generate and manage Apple Wallet passes
- Example scripts for pass creation and distribution
- Ready-to-use templates for common pass types
- Integration with Google and Apple Wallet APIs
